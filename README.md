Once you finish the setup, the agent runs in your system tray. It stays active while you work. When you use your primary coding tool, remora-cc intercepts the outgoing traffic. It analyzes the model request and routes it based on your pre-set rules.

If you notice a request delay, check your internet connection. You can toggle the routing status by right-clicking the icon in the tray and choosing "Pause" or "Resume."

Run only one instance at a time to prevent conflicts with your network ports. If you encounter issues, restart the application from the shortcut.

**How do I uninstall the program?**
Open your Windows Control Panel. Select "Programs and Features." Find remora-cc in the list, right-click it, and select "Uninstall." This removes the application and all associated configuration files from your system.
